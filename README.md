# Aprofundamento em MVC

1. Gerar as pastas do rails:
```bash
  rails new mvc
```
2. Gerando modelos:
```bash
  rails g model product name price:decimal quantidade:integer
```
3. Gerando migrate:
```bash
  rails db:migrate
```
4. Gerando o controller:
```bash
  rails g controller products
```
5. Rails console (inserir dados nos modelos)
```bash
  rails console
```
